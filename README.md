Пинг Понг игра на _python_ с использованием библиотеки **PyGame**.
Об игре:
-Игра на двоих в которой с помощью 2ух ракеток необходимо отбивать шарик
Управление:
-Клавиша **W** - вверх
-Клавиша **S** - вниз
